# USER - 家庭环境与成员画像

## 基本信息

- 通信方式：QQ 或网页
- 时区：东8区 (UTC+8)
- 语言：中文

## 物理空间与设备映射
> 家庭信息从 `data/homes.json` 动态加载，无需手动维护，current=true是当前设备管理的家庭
> 房间信息从 `data/spaces.json` 动态加载，无需手动维护
> 设备信息从 `data/devices.json` 动态加载，无需手动维护，online=false是设备不在线，不要操作这些设备
## 家庭成员

> 成员信息通过日常交互自动收集与优化
>
> 收集内容包括：
> - 作息习惯（起床/就寝时间）
> - 温度/灯光偏好
> - 特殊需求或限制
记录到memory/members.md 中

## 关键场景偏好

### 回家模式
- **触发条件**：指纹锁解锁 + 时间在 18:00-23:00
- **执行动作**：
  - 打开玄关灯和客厅主灯
  - 播放轻音乐

### 观影模式
- **触发条件**：电视开启 + 时间在 19:00 后
- **执行动作**：
  - 客厅灯光调暗至 10%
  - 色温转暖 (3000K)

### 离家模式
- **触发条件**：所有手机蓝牙信号不在家 + 持续 10 分钟
- **执行动作**：
  - 关闭所有灯光
  - 关闭空调
  - 检查门窗状态

### 儿童保护模式
- **规则**：
  - 21:30 后儿童房灯光自动调暗
  - 20:00 后禁止开启电视（需家长确认）

## 确认级别

| 操作类型 | 确认要求 |
|---------|----------|
| 单个灯光/风扇 | 无需确认 |
| 空调温度调整 | 无需确认 |
| 批量设备操作 | 需要确认 |
| 安全设备（门锁等） | 必须确认 |
| 摄像头查看 | 必须确认 |
